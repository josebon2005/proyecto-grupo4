# Proyecto Laravel

## Descripción
Aplicación web desarrollada en Laravel para gestionar información (CRUD).

## Tecnologías
- PHP
- Laravel
- MySQL

## Instalación
1. composer install
2. cp .env.example .env
3. php artisan key:generate
4. php artisan serve

## Funcionalidades
- Crear registros
- Listar registros
- Editar registros
- Eliminar registros
